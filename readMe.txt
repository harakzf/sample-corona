■本プロジェクトの実行方法
$ cd ./sample-corona（sample-coronaディレクトリ直下に移動）
$ python manage.py


■Python仮想環境の作成方法
※前提：ローカル環境にPython3.Xがインストールされていること。

$ cd ./sample-corona（sample-coronaディレクトリ直下に移動）
$ python -m venv venv
$ cd ./venv/Scripts
$ activate
$ cd ../../
$ pip install -r requirements.txt
